# 포커스 특수키

1. 포커스 범위 한 단계씩 늘리기

   (메소드에 괄호가 여러개씩 추가 될때  어디까지 떼어내야할지 애매할때 사용)

    - 맥: `Alt + 위/아래 화살표`
    - 윈도우, 리눅스
        - `Ctrl + W` (위)
        - `Shift + Ctrl + W` (아래)
2. 포커스 뒤로/앞으로 가기
    - 맥: `Command + [,]`
    - 윈도우, 리눅스: `Ctrl + Alt 좌, 우`
3. 멀티 포커스
    - 맥: `Alt + Alt +아래`
    - 윈도우, 리눅스: `Ctrl + Ctrl + 아래`
4. 오류 라인 자동 포커스
    - 맥: `F2`
    - 윈도우, 리눅스: `F2`
