# 리팩토링 Extract

1. 변수 추출하기(Variable)
    - 맥: `Command + Option + V`
    - 윈도우, 리눅스: `Ctrl + Alt +V`
2. 파라미터 추출하기(Parameter)
    - 맥: `Command + Option + P`
    - 윈도우, 리눅스: `Ctrl + Alt + P`
3. 메소드 추출하기(Method)
    - 맥: `Command + Option + M`
    - 윈도우, 리눅스: `Ctrl + Alt + M`
4. 이너클래스 추출하기
    - 맥: `F6`
    - 윈도우, 리눅스: `F6`
