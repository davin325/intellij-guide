# 메인메소드 생성 및 실행

1. 디렉토리, 패키지, 클래스 등 생성 목록 보기
    - 맥: `Command + n`
    - 윈도우, 리눅스: `Alt + Insert`

2. 코드 템플릿
    - 메인 메소드: `psvm`
    - System.out.printn(): `sout`

3. 실행환경 실행
    1. 현재 포커스
        - 윈도우, 리눅스: `Shift + Ctrl + F10`
        - 맥: `Ctrl + Shift + R`
    2. 이전 실행
        - 맥: `Ctrl + R`
        - 윈도우, 리눅스: `Shift + F10`
