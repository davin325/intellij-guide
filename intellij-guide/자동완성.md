# 자동완성

1. 스마트 자동완성
    - 맥: `Ctrl + Shift + Space`
    - 윈도우, 리눅스: `Ctrl + Shift + Space`
    - `Ctrl + Space` 일반 자동완성은 불필요한 경우도 보여주기 때문에 스마트 자동완성을 더 많이 쓴다.
    

2. 스태틱 메소드 자동 완성
    - 맥: `Ctrl + Space * 2`
    - 윈도우, 리눅스: `Ctrl + Space * 2`
    - assertThat 같은 스태틱 메소드 선언시 사용
    

3. Getter / Setter / 생성자 자동완성
    - 맥: `Command + N`
    - 윈도우, 리눅스: `Alt + Insert`
    

4. Override 메소드 자동완성
    - 맥: `Ctrl + I`
    - 윈도우, 리눅스: `Ctrl + I`
