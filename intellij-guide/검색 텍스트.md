# 검색 텍스트
1. 현재 파일에서 검색
    - 맥: `Command + F`
    - 윈도우, 리눅스: `Ctrl + F`
   

2. 현재 파일에서 교체
    - 맥: `Command + R`
    - 윈도우, 리눅스: `Ctrl + R`
   

3. 전체에서 검색
    - 맥: `Command + Shift + F`
    - 윈도우, 리눅스: `Ctrl + Shift + F`
   

4. 전체에서 교체
    - 맥: `Command + Shift + R`
    - 윈도우, 리눅스: `Ctrl + Shift + R`
   

5. 정규표현식으로 검색, 교체
    - 맥: `Regex 체크`
    - 윈도우, 리눅스: `Regex 체크`
