# Git 기본 기능 사용하기

1. Git View On
    - 맥: `Command + 9`
    - 윈도우, 리눅스: `Alt + 9`
2. Git Option Popup
    - 맥: `Ctrl + V`
    - 윈도우, 리눅스: Alt + `(Tab 위 버튼)
3. Git History
    - 맥: `Ctrl + V ⇒ 4`
    - 윈도우, 리눅스: Alt + `(Tab 위 버튼)  => 4
4. Branch
    - 맥: `Ctrl + V ⇒ 7`
    - 윈도우, 리눅스: Alt + `(Tab 위버튼) => 7
5. Commit
    - 맥: `Command + k`
    - 윈도우, 리눅스: `Ctrl + k`
6. Push
    - 맥: `Command + Shift + k`
    - 윈도우, 리눅스: `Ctrl + Shift + k`
    - 파일이 새로 생성된 경우에는 Push를 따로 해주어야 한다.
7. Pull
    - 맥: `Command + Shift + A ⇒ git pull`
    - 윈도우, 리눅스: `Ctrl + Shift + A ⇒ git pull`
