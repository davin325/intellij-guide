# 포커스 에디터

1. 단어별 이동
    - 맥: `Alt + <,>`
    - 윈도우, 리눅스: `Ctrl + <,>`
   

2. 단어별 선택
    - 맥: `Shift + Alt +<,>`
    - 윈도우, 리눅스: `Shift + Ctrl + <,>`
   

3. 라인 첫/끝 이동
    - 맥: `Fn + <,>`
    - 윈도우, 리눅스: `Home, End`
   

4. 라인 전체 선택
    - 맥: `Shift + Command + <,>`
    - 윈도우, 리눅스: `Shift + Home, End`
   

5. Page Up/Down
    - 맥: `Fn + 위/아래`
    - 윈도우, 리눅스: `Page Up / Down`
